# Bilder för Fine Line galleri

Denna mapp ska innehålla alla bilder för webbplatsen.

## Struktur

```
images/
├── favicon.svg              # Favicon (redan inkluderad)
├── artist-placeholder.jpg   # Bild på konstnären (startsida)
├── artist-portrait.jpg      # Bild på konstnären (om-sidan)
├── studio-work.jpg          # Bild från ateljén
├── technique-detail.jpg     # Närbild på teknik
├── category-animals.jpg     # Kategori: Djur
├── category-portraits.jpg   # Kategori: Porträtt
├── category-nature.jpg      # Kategori: Natur
└── artworks/
    ├── lion-savanna.jpg
    ├── woman-profile.jpg
    ├── flamingo-dance.jpg
    └── ...etc
```

## Rekommenderade storlekar

- **Konstverksbilder:** 800x1000px (4:5 ratio) för optimal display
- **Kategori-bilder:** 600x800px (3:4 ratio)
- **Porträttbilder:** 600x750px (4:5 ratio)

## Format

- Använd **.jpg** för fotografier och konstverksbilder
- Använd **.png** för bilder med transparens
- Använd **.svg** för ikoner och logotyper

## Tips

- Komprimera bilder innan uppladdning (t.ex. med TinyPNG/TinyJPG)
- Använd beskrivande filnamn (kebab-case)
- Lägg till alt-texter i HTML för tillgänglighet
